# Rock Paper Scissors Game

A fun and interactive Rock Paper Scissors game built with **HTML**, **CSS**, and **JavaScript**. Play against the computer and be the first to reach 10 points to win!

## 🎮 Features

- **Interactive Gameplay** - Click buttons to play Rock, Paper, or Scissors
- **Automatic Computer Choice** - Computer makes random selections
- **Real-Time Score Tracking** - See both your and computer's scores
- **Visual Feedback** - Large circular displays show choices with animations
- **First to 10 Wins** - Game ends when someone reaches 10 points
- **Beautiful UI** - Modern gradient design with smooth animations
- **Responsive Design** - Works on desktop and mobile devices
- **Reset Button** - Start a new game anytime

## 🎯 Game Rules

- **Rock** beats **Scissors**
- **Scissors** beats **Paper**
- **Paper** beats **Rock**
- First player to reach **10 points** wins the game!

## 📁 Files

- `index.html` - Game structure and layout
- `style.css` - Beautiful styling and animations
- `script.js` - Game logic and interactivity

## 🚀 How to Play

1. Open `index.html` in your web browser
2. Choose your move:
   - 🪨 Rock
   - 📄 Paper
   - ✂️ Scissors
3. The computer automatically makes its choice
4. Your score increases if you win, computer's score if it wins
5. First to reach 10 points wins!
6. Click **Reset** to play again

## 📊 Game Layout

```
        YOUR SCORE
        
    [Your Choice] [Computer Choice]
    
       Game Result
       
    [Rock] [Paper] [Scissors]
    
      COMPUTER SCORE
      
       [Reset Button]
```

## 🔧 Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/MedaAbhinav/Rock-Paper-Scissors.git
   ```

2. Navigate to the directory:
   ```bash
   cd Rock-Paper-Scissors
   ```

3. Open `index.html` in your browser or use a local server:
   ```bash
   python -m http.server 8000
   ```
   Then visit `http://localhost:8000`
